Wallet API
Сервис для управления балансом кошельков.
Реализован на FastAPI с асинхронной работой с PostgreSQL и поддержкой конкурентных операций (блокировка строк на уровне БД).

Проект полностью контейнеризирован и запускается одной командой через Docker Compose.

Технологии
Python 3.11

FastAPI – веб-фреймворк

SQLAlchemy (async) – ORM

PostgreSQL 15 – база данных

Alembic – миграции

pytest – тестирование

Docker / Docker Compose – контейнеризация

Запуск
1.Клонируйте репозиторий
git clone <url-репозитория>
cd wallet_app

2.Запустите контейнеры
docker-compose up --build
После сборки приложение будет доступно по адресу:
http://localhost:8000

3.Документация API (Swagger UI)
Откройте в браузере:
http://localhost:8000/docs

Тестирование
Запуск тестов внутри контейнера:
docker-compose exec app pytest -v

Эндпоинты
1. Получение баланса
GET /api/v1/wallets/{wallet_uuid}

2. Операция с кошельком (пополнение / снятие)
POST /api/v1/wallets/{wallet_uuid}/operation

Особенности реализации
Асинхронность – все эндпоинты работают асинхронно, что обеспечивает высокую пропускную способность.

Блокировка строк – для предотвращения гонок при параллельных запросах используется SELECT ... FOR UPDATE. Это гарантирует корректность баланса даже при конкурентных операциях.

Миграции – управляются через Alembic и применяются автоматически при запуске контейнера.

Тесты – синхронные, с использованием TestClient и psycopg2, что исключает конфликты циклов событий и делает их надёжными.

Примечания
Для создания тестового кошелька вручную можно выполнить SQL-запрос внутри контейнера БД:
docker-compose exec db psql -U postgres -d wallet_db

INSERT INTO wallets (id, balance) VALUES ('123e4567-e89b-12d3-a456-426614174000', 0.00);

Все переменные окружения (например, строка подключения) уже заданы в docker-compose.yml и не требуют дополнительной настройки.
